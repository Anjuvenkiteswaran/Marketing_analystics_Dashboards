# E-commerce Profitability Dashboard

## What is the Business Problem?
Sales are increasing, but profits are inconsistent across products and regions. Which is also called the growth paradox where the costs are increasing compared to the revenue

## What would be the Objective then?
Analyze:
- 1)revenue trends
- 2)customer behavior
- 3)product profitability
- 4)regional performance

## Tools I would be using are-
- SQL
- Power BI
- DAX
